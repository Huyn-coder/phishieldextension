üì¶ Installation Instructions

  1. Download or clone this project to your computer
  2. Open Google Chrome browser
  3. Navigate to chrome://extensions/
  4. Enable "Developer mode" using the toggle in the top right corner
  5. Click the "Load unpacked" button
  6. Select the folder containing this extension (the folder with manifest.json)
  7. The Link Collector extension should now appear in your extensions list
  8. Pin the extension to your toolbar for easy access (click the puzzle icon and pin)

üöÄ How to Use

  1. Browse normally - The extension works automatically in the background
  2. Switch tabs - Each time you switch to a new tab, links are automatically collected
  3. Check your stats - Click the extension icon to see how many links have been collected
  4. Download links - Click "Download Links" in the popup to save all collected links to a text file
  5. Clear when needed - Use the "Clear All Links" button to start a new collection
